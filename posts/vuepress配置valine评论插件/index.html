vuepress配置Valine评论插件

2018.08.19

1. 注册账号

前往https://leancloud.cn/注册账号

2. 获取appId和appKey

路径 应用>设置>应用Keys

3.配置config.js

在themeConfig节点下配置自己Valine信息

module.exports = {
    ......
    "themeConfig": {
        ......
        valineConfig: {
            appId: 'xxxxxxxxxxxxxxxxxxxxxxx',// your appId
            appKey: 'xxxxxxxxxxxxxxxxxxxxxxx', // your appKey
        }
    },
    ......
}
